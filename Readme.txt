# Old Age Home Management System  

A web-based system built using PHP and MySQL to efficiently manage the operations of an old age home.

## 🚀 How to Run the Project  

### Prerequisites  
Ensure you have the following installed on your system:  
- [XAMPP](https://www.apachefriends.org/) / [WAMP](http://www.wampserver.com/) / [LAMP](https://doc.ubuntu.com/lamp/en/)  
- Web browser  
- PHP 7.4 or higher  
- MySQL  

### 📖 Installation Steps  

1. **Download the Source Code**  
   - Clone the repository or [Download the ZIP](#).  

   ```bash
   git clone <repository-url>
   ```

2. **Extract and Copy Files**  
   - Extract the downloaded ZIP file.  
   - Copy the `oahms` folder.  

3. **Paste Inside Root Directory**  
   - For XAMPP: `xampp/htdocs`  
   - For WAMP: `wamp/www`  
   - For LAMP: `/var/www/html`  

4. **Setup Database**  
   - Open [PHPMyAdmin](http://localhost/phpmyadmin)  
   - Create a new database: `oahmsdb`  
   - Import the SQL file located inside the `SQL` folder: `oahmsdb.sql`  

5. **Run the Application**  
   - Open your browser and visit: [http://localhost/oahms](http://localhost/oahms)
